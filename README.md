# ClojureScript tetris
Compiles to JavaScript.
Playable on browser and controllable with keyboard: https://invertisment.github.io/cljs-tetris/

#### Controls:
    Piece control: w, a, s, d
    Hard-drop the current piece: Space
    New game: n

#### Development
##### Tools
lein, cljs, speclj

##### Auto build
lein cljsbuild auto

##### Running tests
lein spec -a
